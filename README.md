## Project Overview

The *Family Expense Management System* is developed on the ServiceNow platform to simplify and automate daily expense tracking within a family.  
It enables users to record, categorize, and analyze expenses efficiently, offering insights into spending patterns through intuitive dashboards and reports.

## Key Features

- User-friendly interface for expense entry and updates  
- Automated categorization and reporting of expenses  
- Real-time data tracking using ServiceNow tables and relationships  
- Business rules for automatic numbering and validation  
- Dashboard for visual analytics and summaries  


## Technologies Used

- *Platform:* ServiceNow  
- *Modules:* Family Expenses, Daily Expenses  
- *Features Used:* Tables, Business Rules, Related Lists, Form Design, Workflow Configuration
